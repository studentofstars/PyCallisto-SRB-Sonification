# PyCallisto-SRB-Sonification

Audio-visual sonification of solar radio burst spectrograms using data from the
[e-CALLISTO](http://www.e-callisto.org/) network.

This repository provides two complementary modules:

| File | Purpose |
|------|---------|
| `callisto_pipeline.py` | Load, join, process and plot e-Callisto FITS spectrograms |
| `spectrogram_sonification.py` | Convert spectrogram data into synchronised audio + animated video |

---

## Background

The [e-CALLISTO](http://www.e-callisto.org/) network is a global array of solar
radio spectrometers that continuously monitors the Sun across 45–870 MHz.
This project processes CALLISTO FITS data using the
[pyCallisto](https://github.com/ravipawase/pyCallisto) library and adds an
audio-visual sonification layer — mapping radio frequency vs intensity into
musical tones to make solar radio bursts *audible*.

The scientific basis for pyCallisto is described in:

> Pawase, R. & Raja, K. S. (2020). *pyCallisto: A Python Library To Process
> The CALLISTO Spectrometer Data.* arXiv:2006.16300 [astro-ph.IM]
> https://arxiv.org/abs/2006.16300

## Installation

```bash
git clone https://github.com/studentofstars/PyCallisto-SRB-Sonification.git
cd PyCallisto-SRB-Sonification
pip install -r requirements.txt
```

### Dependencies

| Package | Purpose |
|---------|---------|
| `pyCallisto` | FITS loading, background subtraction, slicing, plotting |
| `numpy` | Numerical arrays |
| `matplotlib` | Spectrogram and animation plotting |
| `scipy` | WAV file writing |
| `pydub` | Optional: MP3 background loading |
| `ffmpeg` (system) | Optional: MP4 video export |

Install system ffmpeg:
```bash
# Ubuntu / Debian
sudo apt install ffmpeg

# macOS
brew install ffmpeg
```

## Quick Start

### 1 — Process and plot a spectrogram

```python
from callisto_pipeline import run_pipeline

fits_files = [
    "GREENLAND_20170906_115514_62.fit.gz",
    "GREENLAND_20170906_121014_62.fit.gz",
    "GREENLAND_20170906_122514_62.fit.gz",
]

obs = run_pipeline(
    fits_paths=fits_files,
    subtract_bg=True,
    time_slice=("11:56:00", "13:10:00"),   # or None
    freq_slice=(10, 100),                   # MHz, or None
    xtick=8,
)
```

### 2 — Sonify a processed spectrogram

```python
from spectrogram_sonification import play_spectrogram_with_audio

play_spectrogram_with_audio(
    obs,
    mapping_mode='pentatonic',   # see Musical Scales section below
    audio_duration=15,
    save_video=True,
    video_filename="solar_radio_chimes_20170906.mp4",
)
```

### 3 — Additional analysis

```python
from callisto_pipeline import plot_light_curve, plot_mean_spectrum, plot_universal

plot_light_curve(obs)         # mean flux vs time
plot_mean_spectrum(obs)       # mean flux vs frequency
plot_universal(obs, title="20170906 Type III Burst")  # combined 3-panel plot
```

## Musical Scale Modes

The sonification maps radio frequency bins to audible tones using one of several
musical scales:

| `mapping_mode` | Scale | Character |
|---|---|---|
| `pentatonic` | Major pentatonic | Bright, no dissonance |
| `pentatonic_minor` | Minor pentatonic | Darker, introspective |
| `major` | Major (Ionian) | Classic, uplifting |
| `minor` | Natural minor (Aeolian) | Melancholic |
| `blues` | Blues with blue note | Expressive, jazzy |
| `japanese` | Hirajoshi | Sparse, meditative |
| `auld_lang_syne` | Major pentatonic variant | Nostalgic |
| `proportional` | Log-linear mapping | Preserves radio structure |

## Configuration

Edit the constants at the top of each file, or pass keyword arguments:

**`callisto_pipeline.py`**

```python
FITS_PATHS      = [...]          # list of .fit / .fit.gz files
TIME_SLICE      = ("HH:MM:SS", "HH:MM:SS")  # or None
FREQ_SLICE      = (f_min, f_max)             # MHz, or None
SUBTRACT_BG     = True
XTICK           = 5              # minute interval for x-axis ticks
BLEVEL          = 0              # colour floor
VMAX            = 40             # colour ceiling
```

**`spectrogram_sonification.py`**

```python
MAPPING_MODE        = 'pentatonic'
AUDIO_DURATION      = 15         # seconds
INTENSITY_THRESHOLD = 0.35       # 0.0–1.0; only sonify above this level
MIN_AUDIBLE_FREQ    = 800        # Hz
MAX_AUDIBLE_FREQ    = 4000       # Hz
SAVE_VIDEO          = True
VIDEO_FILENAME      = "solar_radio_chimes.mp4"
USE_CUSTOM_MP3      = False      # mix a background MP3
CUSTOM_MP3_PATH     = ""
```
